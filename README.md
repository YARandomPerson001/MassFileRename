# MassFileRename
A script to mass rename files in a directory.

## Flags
### Auto Yes
* -y
* --y
* -yes
* --yes

### Directory to mass rename
* -d
* --d
* -dir
* --dir

### The word that gets to replace
* -p
* --p
* -part
* --part

### The replacement word
* -r
* --r
* -replace
* --replace

### For a empty string
* -e
* --e
* -empty
* --empty

### Help
* -h
* --h
* -help
* --help
